# CV Web — Javier Díaz (HTML + CSS)

Web minimalista para currículum, lista para publicar en GitHub Pages. Pruebas de cambio desde VSC -> GitHUB

## 1) Pon tu foto
- Crea la carpeta `assets/` si no existe
- Mete tu foto como: `assets/profile.jpg`

> Consejo: usa una foto cuadrada (p.ej. 900x900). Si no, se verá igualmente bien.

## 2) Edita enlaces y contacto
Abre `index.html` y cambia:
- Email
- LinkedIn
- GitHub
- Textos de proyectos (Demo/GitHub)

## 3) Publicar en GitHub Pages
1. Crea un repo nuevo en GitHub (por ejemplo: `cv` o `javierdiaz-cv`)
2. Sube estos archivos al repo (en la raíz)
3. En GitHub: Settings → Pages
   - Source: Deploy from a branch
   - Branch: main / root
4. Guarda y te dará la URL:
   - `https://TUUSUARIO.github.io/NOMBRE-REPO/`

Si quieres que sea `https://TUUSUARIO.github.io/`:
- El repo debe llamarse exactamente `TUUSUARIO.github.io`

## Extra: dominio propio
Si compras un dominio, puedes apuntarlo a GitHub Pages y añadir un `CNAME`.
